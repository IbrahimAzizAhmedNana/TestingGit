Easy Git Commands

Set Up Git:
git config --global user.name "Your Name"
git config --global user.email youremail@domain.com

other commands
1) git clone (url) // this is where you clone/copy the source code from the repo in the link
2) cd (repo-name) // you need to do this to add/update 

3) git add (file names) // say which files you hava either added or updated
4) git commit -m "(committed message) // a revison or adding changes to the local repo
5) git push -u origin master // transfer last commit to a remote server
6) git remote add origin master (url)
